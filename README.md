# 金庸武侠DND肉鸽踢门团

一个给 AstrBot 使用的群聊文字肉鸽插件。玩家选择金庸武侠门派开局，在武神塔中逐层踢门，遇到战斗、宝箱、奇遇、陷阱、商人等随机事件，通过公开 d20 检定推进流程，最终挑战第 7 层武神殿的武神镜像。

插件目标是把 DND5e 的“公开骰子检定”和金庸武侠门派成长结合成适合群聊游玩的轻量跑团小游戏。

## 功能特性

- 16 大门派开局
- 普通/困难两档难度
- 1~6 层武神塔，每层 3 个随机事件门
- 第 7 层武神殿挑战武神镜像
- 标准 d20 检定，骰点、加值、DC 全部明示
- 本门奇遇、残页、补给、碎银、素材、全局奖励
- 钓鱼模块：钓点、饵剂、稀有度奖励
- 通关后角色飞升冻结
- 无第三方 Python 依赖，使用 JSON 保存玩家档案

## 安装

进入 AstrBot 根目录，将本仓库放入 `data/plugins/`：

```bash
git clone https://github.com/Rudyy898/astrbot_plugin_jinyong_rogue.git data/plugins/astrbot_plugin_jinyong_rogue
```

重启 AstrBot 后启用插件。

## 指令

| 指令 | 说明 |
| --- | --- |
| `/jy帮助` | 查看游戏指令 |
| `/jy门派` | 查看 16 大门派 |
| `/jy开局 门派 [普通\|困难]` | 创建角色 |
| `/jy状态` | 查看当前角色 |
| `/jy踢门` | 开启当前层随机事件门；第 7 层挑战武神镜像 |
| `/jy下一层` | 清完 3 门后触发本门奇遇并进入下一层 |
| `/jy钓鱼 [钓点] [饵剂]` | 进行钓鱼，默认山涧浅滩+普通蚯蚓饵 |
| `/jy钓鱼 列表` | 查看钓点和饵剂 |
| `/jy重置 confirm` | 删除当前角色档案 |

## 快速开始

```text
/jy门派
/jy开局 华山 普通
/jy状态
/jy踢门
/jy踢门
/jy踢门
/jy下一层
```

钓鱼示例：

```text
/jy钓鱼
/jy钓鱼 寒潭深水区 秘制香饵
/jy钓鱼 秘境水潭 龙涎香饵
```

## 游戏规则

### 武神塔

武神塔共 7 层：

- 第 1~6 层：每层固定 3 个事件门
- 第 7 层：武神殿终极挑战
- 清完当前层 3 个门后，使用 `/jy下一层` 触发一次本门奇遇检定
- 第 7 层使用 `/jy踢门` 挑战武神镜像

事件门不会提前生成，只有玩家发送 `/jy踢门` 时才生成并结算。

### 事件类型

| 类型 | 权重 | 说明 |
| --- | ---: | --- |
| 战斗 | 40% | 门派主属性 d20 检定，胜利获得碎银和素材 |
| 宝箱 | 20% | 获得碎银、补给、武学残页，可能触发机关 |
| 奇遇 | 15% | 高人指点，恢复或获得素材 |
| 陷阱 | 15% | 拆解成功得奖励，失败损失 HP |
| 商人 | 10% | 交涉购买补给 |

### 难度

| 难度 | 奇遇 DC | 敌人增幅 | 通关奖励 |
| --- | ---: | ---: | --- |
| 普通 | 10 | +0 | 武道真髓 x2、武学残卷 x5 |
| 困难 | 13 | +3 | 武道真髓 x4、武学残卷 x10 |

### 钓鱼

钓点：

- 山涧浅滩
- 寒潭深水区
- 秘境水潭
- 门派专属钓点
- 武神塔顶龙池

饵剂：

- 普通蚯蚓饵
- 秘制香饵
- 龙涎香饵

## 数据保存

玩家档案保存到 AstrBot 插件数据目录下：

```text
players.json
```

当前版本按用户 ID 保存角色。角色通关后会进入“飞升冻结”状态，符合蓝本中“单角色通关飞升后档案永久冻结”的设定；如需重开，玩家可手动 `/jy重置 confirm`。
